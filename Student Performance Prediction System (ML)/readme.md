# Student Performance Prediction System

> AI/ML Internship Project 2 – CODEVEDX

## Project Description

The **Student Performance Prediction System** is a Machine Learning-based console application developed using Python. It helps predict a student's final marks based on factors such as attendance, study hours, assignment scores, and internal marks.

The application reads student data from a CSV file, trains a Linear Regression model, and predicts the expected final marks for new student inputs. This project demonstrates the practical implementation of Python, data preprocessing, and Machine Learning concepts.

# Machine Learning Algorithm

## Linear Regression

Linear Regression is a supervised Machine Learning algorithm used to predict continuous values. In this project, it learns the relationship between attendance, study hours, assignment scores, internal marks, and final marks to estimate a student's academic performance.
